INFORMATIQUE

Computer science program

This program will be available in 2022, when the equipments will be operational. It's subdivided into the following parts:

  • The basic computing
  • Programming.

1. The basic computing

As the name suggests, this part introduces our learners to the basics of computer science. Some points like "How to turn on a computer" are booked here.

2. Programming

In this part our attention will be drawn to the programming in HTML, CSS and PHP languages. But a little introduction to the programming wouldn't be boring to catch up our learners. Here, our purpose is to make our students webmasters, thus they will be able to conceive dynamic web sites and so on.
